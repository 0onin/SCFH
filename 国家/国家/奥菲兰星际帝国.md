> **“钢铁与火焰中铸就的秩序之刃”**

**奥菲兰星际帝国**是一个以军事效率、社会纪律和工业实力著称的超级文明。其文明基石建立在“绝对秩序带来绝对力量”的信念之上，整个帝国如同一台精密而冷酷的战争机器。

---

# 核心概览

- **全称**：奥菲兰星际帝国
- **政体**：军事贵族集权制
- **首都**：钢铁王座世界「铁王星」 ^71b0e9
- **国家元首**：钢铁大公
- **文明信条**：力量源于秩序，荣誉来自征服
- **实力定位**：超级文明中的军事强国

# 文明特征

奥菲兰人以其严谨、坚韧和对权威的绝对服从而闻名。帝国社会呈现出高度的军事化特征：
- **尚武文化**：军事成就被视为最高荣誉，公民从小接受军事化管理和基础战斗训练。
- **工业美学**：其建筑、舰船乃至日常用品都秉承着厚重、实用、棱角分明的设计风格，充满力量感。
- **效率至上**：社会运转极度追求效率，任何被视为“低效”或“无序”的行为都会受到严厉惩戒。

# 历史沿革

1.  **统一战争时期**：在母星「奥菲兰」的战国时代，**钢铁家族**凭借其强大的军事组织和先进的工业能力，用了三个世纪的时间统一了整个星球，建立了帝国雏形。
2.  **钢铁律法时期**：第一位**钢铁大公**颁布了奠定帝国基础的《钢铁法典》，将军事化管理推广到社会每一个角落，并开启了太空扩张时代。
3.  **星海争霸时期**：在扩张中与埃维斯帝国及其影响势力发生多次冲突。屡次受挫于埃维斯的深不可测后，奥菲兰开始主动与其他感受到埃维斯压力的超级文明进行“非正式协调”，形成了当前抗衡格局。

# 行政区划

帝国实行严格的**军区制**，将疆域划分为五大星域军区，每个军区下辖若干星系集群。
- **核心星域**：环绕铁王星，是帝国的政治、工业和军事研发中心。
- **边境星域**：直面埃维斯影响区域或边界，驻守着帝国最精锐的舰队，享有高度自治的军事指挥权。

# 核心资源

- **物产**：帝国疆域内富含多种稀有金属和放射性元素，是其庞大重工业的基础。
- **人口**：奥菲兰人以人类为主，身体素质普遍强悍，人口增长率受到严格计划控制，以确保资源与人口的平衡。
- **环境**：倾向于改造类地行星为适宜工业生产的“钢铁世界”，星球生态系统通常较为单一和稳定。

# 政治与军事

- **政治体制**：由世袭的**钢铁大公**作为最高统治者，其下是由各大家族和军方巨头组成的**钢铁议会**。决策过程直接且高效，但内部派系围绕资源和新领土的分配存在激烈竞争。
- **军事力量**：帝国海军是其骄傲，以数量庞大、装甲厚重的**无畏舰**和强调正面突击的战术闻名。其标志性的“**钢铁冲击**”战术，能以密集阵形强行撕开敌方防线。

# 经济与社会

- **经济模式**：实行**指令性计划经济**，核心战略资源与重工业由国家完全掌控，民生领域允许有限的市场经济存在。
- **社会结构**：一个清晰的**军事种姓制度**，公民的社会地位与其（或直系亲属）的军功直接挂钩。

# 文化与外交

- **文化**：艺术与文学主题多为歌颂战争、牺牲与集体荣誉。个人主义被抑制，对帝国的忠诚是最高美德。
- **外交**：态度强硬，信奉“真理只在大炮射程之内”。是抗衡联盟中最主张采取强硬军事姿态的成员，对其他盟友既合作又充满竞争意识。
